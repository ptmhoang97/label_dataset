# How to use labelimg to label dataset?
**1) Install:**
```
pip install labelimg
```
**2) Open:**
- Run command:
```
labelimg
```
**3) Set image path:**
- Click "Open Dir"\
![image](https://user-images.githubusercontent.com/53186326/141612656-dfbc3d4e-9d2e-410c-9021-57f55b73b540.png)

**4) Set save path for annotations:**
- Click "Change Save Dir"\
![image](https://user-images.githubusercontent.com/53186326/141612651-453d2247-ca43-49dc-b54a-759c1291f7f2.png)

**5) Change format annotations:**
- Click to button to change format: "YOLO" or "CreateML" or "PascalVOC"\
![image](https://user-images.githubusercontent.com/53186326/141612760-3e00081d-a18f-468d-bcdb-ba1f12b15582.png)
![image](https://user-images.githubusercontent.com/53186326/141612767-07fe8455-89f0-464c-8192-82bf55facd22.png)
![image](https://user-images.githubusercontent.com/53186326/141612772-bd777574-67a3-4b64-a246-534c74022576.png)

**6) Other main feature:**
- Click "View" and choose:\
![image](https://user-images.githubusercontent.com/53186326/141612608-4cd57cab-b499-4a19-8420-30a7ccc5d063.png)

**7) Other hotkey:**
- A: previous image
- D: next image
- W: draw box
